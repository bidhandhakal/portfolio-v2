# Portfolio v2

A modern, responsive portfolio website built with **Next.js 16**, **TypeScript**, and **Tailwind CSS**.

## üöÄ Features

- **Responsive Design**: Optimized for all screen sizes.
- **Dark/Light Mode**: Seamless theme switching using `next-themes`.
- **Experience Timeline**: Detailed professional history with technology tags and project links.
- **Project Showcase**: Interactive project cards with status indicators and tech icons.
- **Command Menu**: Quick navigation and actions via a command palette (`cmdk`).
- **Modern UI**: Built with **Radix UI** (shadcn/ui) and animated with **Framer Motion**.
- **Tooltips**: Enhanced UX with informative tooltips for links and icons.

## üì¶ Getting Started

### Prerequisites

- Node.js 18+
- npm / pnpm / yarn

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/bidhandhakal/portfolio-v2.git
   cd portfolio-v2
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## üìÇ Project Structure

- `src/app`: Next.js App Router pages and layouts.
- `src/components`: Reusable UI components (Hero, Experience, Projects, etc.).
- `src/hooks`: Custom React hooks.
- `src/lib`: Utility functions and configurations.
- `public/`: Static assets (images, fonts, etc.).
